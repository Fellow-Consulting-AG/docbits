# Doğrulanmış Belgeler

Bu videoda, izleyicilere Doc**Bit**'in kullanıcı dostu arayüzü tanıtılırken özellikle belge doğrulamasına odaklanılmaktadır. Kılavuz, sistemin basitliğini vurgular, kodlama veya karmaşık formüllerin gerekli olmadığı bir ortam sunar.



{% embed url="https://www.youtube.com/watch?v=FD9dC2_nvuw" %}

Yapay Zeka ve Sürü Zekası sayesinde, çoğu alan otomatik olarak çıkarılır, bu da kullanıcıların belge içindeki değerlere tıklayarak bilgi çıkarmasını kolaylaştırır. Ayrıca, izleyiciler istenen bölge etrafında çift tıklayarak veya çizerek belirli alanları çıkarma esnekliğini öğrenirler.
