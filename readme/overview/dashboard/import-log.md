# Import Log

If you would like to see a detailed log of all documents imported to your DocBits environment, then follow these steps.

From your DocBits Dashboard, select the Advanced Settings option.

<figure><img src="../../.gitbook/assets/docbits_settings_workflow.png" alt="DocBits Instellingen Workflow"><figcaption></figcaption></figure>

After the Advanced Settings bar appears, select the Import Log button.

<figure><img src="../../.gitbook/assets/docbits_approval_supplier.png" alt="DocBits Goedkeuring Leverancier"><figcaption></figcaption></figure>

A table will then appear with all recently imported documents as well as other relevant information regarding them.

<figure><img src="../../.gitbook/assets/docbits_purchase_order_table.png" alt="DocBits Aankoop Bestelling Tabel"><figcaption></figcaption></figure>
