# WatchDog Yapılandırması

Bu kılavuz, WatchDog'un DocBits uygulaması içinde nasıl yapılandırılacağını açıklar.

## 1. Genel Bakış

DocBits ayarlar menüsünden WatchDog ayarlarına erişin.

WatchDog için giriş ve çıkış dizinlerini yapılandırın. Bu yolların kurulum işlemi sırasında oluşturulan dizinlerle eşleştiğinden emin olun.

<figure><img src="../../.gitbook/assets/watchdog_v2_02.png" alt="WatchDog Klasör Ayarları"><figcaption></figcaption></figure>

Belge türleri ve diğer işleme seçenekleri gibi genel ayarları yapılandırın.

## 4. Dışa Aktarma Ayarları

Dışa aktarma hedeflerini ve yöntemlerini ayarlayın.

<figure><img src="../../.gitbook/assets/watchdog_v2_04.png" alt="WatchDog Dışa Aktarma Ayarları"><figcaption></figcaption></figure>

## 5. Dışa Aktarma Yapılandırma Detayları

Belirli dışa aktarma yapılandırmalarını görüntüleyin ve yönetin.

<figure><img src="../../.gitbook/assets/watchdog_v2_05.png" alt="WatchDog Dışa Aktarma Yapılandırma Detayları"><figcaption></figcaption></figure>

## 6. BOD Eşleme Yapılandırması

Infor ile entegrasyon için BOD eşlemesini yapılandırın.
