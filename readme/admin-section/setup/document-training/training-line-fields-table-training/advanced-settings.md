## Odwrócone grupowanie

Jeśli chcesz połączyć wszystkie wiersze powyżej atrybutu grupowania, zaznacz pole wyboru tutaj.

W tym przykładzie tabela zaczyna się od wiersza, który znajduje się powyżej wszystkich innych informacji, ale także musi zostać wyodrębniony wraz z informacjami poniżej. Może się zdarzyć, że DocBits (DOC²) wyodrębnia ten wiersz jako dodatkowy wiersz i grupowanie informacji, np. według pozycji, nie działa poprawnie.
