# Erweiterte Einstellungen

## Anzahl der Kopfzeilen

Hier können Sie die Anzahl der Zeilen einer Tabellenüberschrift definieren. Zum Beispiel kann die Tabellenüberschrift zwei Zeilen haben:

![](https://lh7-us.googleusercontent.com/J\_nYZKzUSJHcpJuNp1ykf6shnfetOuiIPhyhyTfSqh\_cTsDT5obwOSIc21OsLkjF9tMBP7Q1GQ1ZQfBsEmQIrJpfX3QpdjHnLkzInsjpngeg2L7i\_TAyl9MdVmgHsDpUvoURdxfqp8FOdJxuRAMCwhk)

Entsprechend ist der Wert in "Anzahl der Kopfzeilen" auf zwei gesetzt

![Anzahl der Kopfzeilen](https://lh7-us.googleusercontent.com/G8-QpKxnRin9PGPmkbtJ35r1EugXsD4\_Yd5QvTkdbb7sXqRVC3a9t0cIlNILJHLr--GsykgAlMJmMYbJaUoMqHEPvAT3PlPNf-syCmPz\_GVMFEMsbhcHI2DQBbT4MJXjS2Sx7M3xl4zAzcw3aa-XNms)

Warum ist das nötig? Es könnte sein, dass DocBits die zweite Zeile in der Tabellenüberschrift nicht als Teil der Kopfzeile erkennt. In diesem Fall fügt es sie fälschlicherweise als extrahierten Wert in die Tabelle ein. Dies kann mit dieser Funktion leicht verhindert werden.

![](https://lh7-us.googleusercontent.com/11D-KFXDqc-u5GQeGvTHfxYbbhLYDX12vFHK9Xkv8YmIcBuWVbFw3CMEhzep\_zyQpr2G9bP1Q0G9mfBj6twopQ-2aRfncbBVbTibXoynkxWYsvD9EnGVO-SfRxHwuPxMqXl4lzjuDoSaYop1HLa7biU)

## Umgekehrte Gruppierung

Wenn Sie alle Zeilen über dem gruppierten Attribut kombinieren möchten, aktivieren Sie hier das Kontrollkästchen.

In diesem Beispiel beginnt die Tabelle mit einer Zeile, die über allen anderen Informationen steht, aber auch zusammen mit den darunter liegenden Informationen extrahiert werden muss. Es könnte sein, dass DocBits (DOC²) diese Zeile als zusätzliche Zeile extrahiert und die Gruppierung der Informationen, z. B. nach Position, nicht ordnungsgemäß funktioniert.
