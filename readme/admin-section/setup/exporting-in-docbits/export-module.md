## Nowy eksport

Aby dodać nową konfigurację eksportu, wybierz „+ Nowy” 

![](https://lh7-us.googleusercontent.com/WIgkQYOGRxpcBeA-fFL-uXMfOh-jAf2Tis1rrOdOtZipjkbWEJJ11Osao0SW0lbIoa5xbBePAGWlxZ4iVHu_QjYcwVa04wNegpbWlRDMf-HHLuKU5hN_162pJT0O6WZt7W76LyRehijqPdbDMXtHpvI)

Wybierz metodę, którą chcesz użyć do konfiguracji eksportu.

![](https://lh7-us.googleusercontent.com/DW9Iapybj7gi-YxSgToHXcxUM6XUZm6RfMdf1vjbKqyiG1gwprMQccXRuJ5A3aTPMX1WBbin_YZIRBXa5LjbpiSrZ0WmOfQ6XRozucbYmD49RC33vnQDxxdRY21k3acROm4DYvrdnDhHyN217ROFuCk)

Po wybraniu metody eksportu, będziesz musiał/a przesłać różne informacje i pliki wymagane dla tej metody eksportu.

## Aktywuj/Dezaktywuj

Gdy masz jedną lub wiele konfiguracji eksportu w swoich DocBits, masz opcję aktywowania lub dezaktywowania konfiguracji w zależności od potrzeb.

Poniższa konfiguracja jest aktywowana, oznaczona zielonym kropką po lewej stronie nazwy konfiguracji.

![](https://lh7-us.googleusercontent.com/bzcV62GoOeLQ2Xy7wRBmtmgxtFXCPF-4Mj9bBofhZFdPzyYDdxQ4jhmQ3XUoJ-YnwP3cWYKpg1v9jt39PiAXP-UnslcQ-DSyTpFEEi7PanZCQEp56rpgf0CcOjwUsRV6JgJxrgt1ieztK3heAeVix50)

Aby dezaktywować konfigurację eksportu, wybierz przycisk opcji po prawej stronie konfiguracji, jak pokazano poniżej.

![](https://lh7-us.googleusercontent.com/oZ150Wzf2_nBtR5iEcWCbzUkxKXZbFvUM3hVfUnhk0uLIcoLumRAPR9DoieW9sdD6ZES1_IWxguRbyFiExe1fMtR_Tg5zRUcLjDllxGnLAtnpyFzi96Ac-xCta3HWDYlW4I0TU_zJb9PPJMGUtC7dbw)

Masz trzy opcje

* Dezaktywuj: Konfiguracja eksportu przestanie być funkcjonalna (oznaczone czerwoną kropką obok nazwy konfiguracji).
* Edytuj: Dokonaj zmian w szczegółach konfiguracji.
* Usuń: Usuń konfigurację.
