## Nouvel export

Pour ajouter une nouvelle configuration d'export, sélectionnez "+ Nouveau" 

![](https://lh7-us.googleusercontent.com/WIgkQYOGRxpcBeA-fFL-uXMfOh-jAf2Tis1rrOdOtZipjkbWEJJ11Osao0SW0lbIoa5xbBePAGWlxZ4iVHu_QjYcwVa04wNegpbWlRDMf-HHLuKU5hN_162pJT0O6WZt7W76LyRehijqPdbDMXtHpvI)

Sélectionnez la méthode que vous souhaitez utiliser pour votre configuration d'export.

![](https://lh7-us.googleusercontent.com/DW9Iapybj7gi-YxSgToHXcxUM6XUZm6RfMdf1vjbKqyiG1gwprMQccXRuJ5A3aTPMX1WBbin_YZIRBXa5LjbpiSrZ0WmOfQ6XRozucbYmD49RC33vnQDxxdRY21k3acROm4DYvrdnDhHyN217ROFuCk)

Une fois que vous avez sélectionné la méthode que vous souhaitez utiliser, vous devrez télécharger les différentes informations et fichiers requis pour cette méthode d'exportation.

## Activer/Désactiver

Une fois que vous avez une ou plusieurs configurations d'export dans vos DocBits, vous avez la possibilité d'activer ou de désactiver les configurations en fonction de vos besoins.

La configuration ci-dessous est activée, indiquée par le point vert à gauche du nom de la configuration.

![](https://lh7-us.googleusercontent.com/bzcV62GoOeLQ2Xy7wRBmtmgxtFXCPF-4Mj9bBofhZFdPzyYDdxQ4jhmQ3XUoJ-YnwP3cWYKpg1v9jt39PiAXP-UnslcQ-DSyTpFEEi7PanZCQEp56rpgf0CcOjwUsRV6JgJxrgt1ieztK3heAeVix50)

Pour désactiver la configuration d'export, sélectionnez le bouton d'options à droite de la configuration comme indiqué ci-dessous.

![](https://lh7-us.googleusercontent.com/oZ150Wzf2_nBtR5iEcWCbzUkxKXZbFvUM3hVfUnhk0uLIcoLumRAPR9DoieW9sdD6ZES1_IWxguRbyFiExe1fMtR_Tg5zRUcLjDllxGnLAtnpyFzi96Ac-xCta3HWDYlW4I0TU_zJb9PPJMGUtC7dbw)

Vous avez trois options

* Désactiver : La configuration d'export ne sera plus fonctionnelle (indiquée par un point rouge à côté du nom de la configuration).
* Modifier : Apportez des modifications aux détails de la configuration.
* Supprimer : Supprimez la configuration.
