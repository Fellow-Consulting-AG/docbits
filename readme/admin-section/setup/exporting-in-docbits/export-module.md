## Nueva Exportación

Para agregar una nueva configuración de exportación, selecciona “+ Nuevo” 

![](https://lh7-us.googleusercontent.com/WIgkQYOGRxpcBeA-fFL-uXMfOh-jAf2Tis1rrOdOtZipjkbWEJJ11Osao0SW0lbIoa5xbBePAGWlxZ4iVHu_QjYcwVa04wNegpbWlRDMf-HHLuKU5hN_162pJT0O6WZt7W76LyRehijqPdbDMXtHpvI)

Selecciona el método que deseas utilizar para tu configuración de exportación.

![](https://lh7-us.googleusercontent.com/DW9Iapybj7gi-YxSgToHXcxUM6XUZm6RfMdf1vjbKqyiG1gwprMQccXRuJ5A3aTPMX1WBbin_YZIRBXa5LjbpiSrZ0WmOfQ6XRozucbYmD49RC33vnQDxxdRY21k3acROm4DYvrdnDhHyN217ROFuCk)

Una vez que hayas seleccionado el método que deseas utilizar, deberás cargar la información y archivos necesarios para ese método de exportación.

## Activar/Desactivar

Una vez que tengas una o varias configuraciones de exportación en tus DocBits, tienes la opción de activar o desactivar las configuraciones según tus necesidades.

La configuración a continuación está activada, indicada por el punto verde a la izquierda del nombre de la configuración.

![](https://lh7-us.googleusercontent.com/bzcV62GoOeLQ2Xy7wRBmtmgxtFXCPF-4Mj9bBofhZFdPzyYDdxQ4jhmQ3XUoJ-YnwP3cWYKpg1v9jt39PiAXP-UnslcQ-DSyTpFEEi7PanZCQEp56rpgf0CcOjwUsRV6JgJxrgt1ieztK3heAeVix50)

Para desactivar la configuración de exportación, selecciona el botón de opciones a la derecha de la configuración como se muestra a continuación.

![](https://lh7-us.googleusercontent.com/oZ150Wzf2_nBtR5iEcWCbzUkxKXZbFvUM3hVfUnhk0uLIcoLumRAPR9DoieW9sdD6ZES1_IWxguRbyFiExe1fMtR_Tg5zRUcLjDllxGnLAtnpyFzi96Ac-xCta3HWDYlW4I0TU_zJb9PPJMGUtC7dbw)

Se te ofrecen tres opciones

* Desactivar: La configuración de exportación dejará de ser funcional (indicado por un punto rojo junto al nombre de la configuración).
* Editar: Realizar cambios en los detalles de la configuración.
* Eliminar: Eliminar la configuración.
