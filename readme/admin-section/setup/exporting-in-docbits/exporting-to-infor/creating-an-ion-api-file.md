# Kreiranje ION API datoteke

## Dobijanje postojeće/Kreiranje nove ION API datoteke

Ovo se kreira u INFOR ION API → Autorizovane aplikacije i trebalo bi da bude prikazano nešto slično kao što je prikazano ispod

![](https://lh7-us.googleusercontent.com/Lpl5VZkrSk9u04C7SWTmfDVbQpRTxSMEE3i2S2Ry\_TDZ9mYJMt6UkzTPw5ApaVIEhzpUmpnffntqXowMlhib8LqIRWgpEYlepZlPyqGFNTTn7z--owAFltp9YfD1Si1SbAAX0c0EHvKrIkb0kK1GboQ)

Ako to nije slučaj, onda morate kreirati novu Autorizovanu aplikaciju. To možete uraditi klikom na znak plus

![](https://lh7-us.googleusercontent.com/y0G9v9WfKlQWCkMy7wmNp59PpUr9AOecJy5zThtVJ5rh09eyH1WkheONuNciZ1pj7-nHKpFSSUjuzYcCe3xE9qrZZcl3IEKeqUWn9h0LSQIF4WJhNDwM9xLaSYskDO4ci2qi3zXPLLl\_nPX8GL2UTK4)

Kada unesete Naziv (DocBits\_\*Okruženje\*) i Tip (specifičan za slučaj) nove Autorizovane aplikacije, bićete preusmereni na stranicu gde su Klijentski ID i Tajna automatski generisani.

Informacije koje popunite trebalo bi da budu slične onima koje su prikazane gore, važno je omogućiti klizač "Izdavanje osvežavajućih tokena" na dnu stranice.

![](https://lh7-us.googleusercontent.com/FevNa7kQ-rCicUxacuCe7m4XHsLOsYYCRq4JAQ3xDTYPxfq3mg5m2BwWF8iz7ASdNS93sgc1t8gJvnNupCDzQPbUu1NpCu4MHxKGBi6OtyDs7ep2TrsaP3gEXHgMsKzHVQ6JbNc2exbdJ41l\_0a7SCA)

Kliknite na "Preuzmi akreditive" da biste preuzeli ION Mapping datoteku.

![](https://lh7-us.googleusercontent.com/rf0BjR-d15LYj98tRJbMClAqtKAqhQdChgsxKS2Ndl-qhquZELEK66WjREvqDcCCj2UlZYK1i-ntNMtzrN\_Kpv1572tqMvWEld8tBDlAzuqz9ocXssiGXFCRnbyH2baWOiBH\_27cpVV3vFUP\_5CJ1xI)

## Postavljanje datoteke na DocBits

Kada preuzmete ION API datoteku iz Infora, možete je postaviti tako što ćete otići na Podešavanja → Obrada dokumenata → Izvoz kao što je prikazano ispod
