# M3

Sobald Sie ein Dokument validiert und exportiert haben, melden Sie sich bei INFOR an und geben Sie den Befehl APS450 ein.

![](https://lh7-us.googleusercontent.com/i4iRIFS78R3roSyp5i6TK8VCp3\_GLgdet13ZyWRgUJjaTkLnl4Kfm-fv\_xky2Pu-LHmvrjLHhv1es\_MpBZAQV0fZPmYa7fsEye9IyNGkLsJNwPIRlSWKZ8BOitnImB6szvGZfaqu-gzVL8vbBhsmpHA)

Sie gelangen zu folgender Seite, auf der Sie das gerade exportierte Dokument sehen.

![](https://lh7-us.googleusercontent.com/zvHZ3f9eVk04gFXYvmCFJZH4zXY9HnuK5BwU-iO7aemf9fVXWDa7Yy6j9F6fIuN0Nx5mlBsRBVbYexe4D547kkgWMMWfEx1sjjEmfORq6cf1gel94LchPvBdPP6htbcCe0bveKRIUlQjr13HPXn49yM)

Sobald Sie Ihr Dokument gefunden haben, klicken Sie mit der rechten Maustaste auf das Dokument und wählen Sie "Verwandte" und dann die Option "Zeilen".

![](https://lh7-us.googleusercontent.com/BTIGc1czaZLPysu1BMTr1hUL\_u0RlnuP0wV7xghtfpyw4L1VEDukpVPJt2VYyD88\_-dQGJ5zaYn4mPKbmJal52-9TwkGzJNBCZ7bgWStiJrWHAIsTGp-BM9TCY328ysf1LJINXKVEOBkdCPFXPxLrzI)

Sie gelangen zum Menü, das alle Posten aus der Rechnung anzeigt, die Sie exportiert haben, um sicherzustellen, dass alle Posten vorhanden und korrekt sind. Stellen Sie abschließend sicher, dass all diese Posten als Typ 8 klassifiziert sind, dies kann durch Betrachten der folgenden Spalte in der Tabelle festgestellt werden.
