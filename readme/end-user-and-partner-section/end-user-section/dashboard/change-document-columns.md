# Wijzig Document Kolommen

Klik vanuit het Dashboard op het pictogram Geavanceerde Instellingen zoals hieronder weergegeven:

<figure><img src="../../../.gitbook/assets/docbits_dashboard_columns_settings.png" alt="Docbits Dashboard Columns Settings"><figcaption></figcaption></figure>

Het volgende menu wordt dan weergegeven:

![Advanced Settings Menu](https://lh7-us.googleusercontent.com/wWt5QbmwZf44enmOoLcofh6SvyYPiHTav9OiEog_m2xtnty6X73pFlhfdM9aglx89_pfbiACZx5BejagV-wAKwlDTuGoGNu5jgbcZ5djrZ_h1IgGp-8uaq8UHY-umjrs96hb4FZOzHFzdLasg2F_ftw)

Selecteer de knop met het label “Stel dashboardkolommen voor organisatie in” en een lijst met alle kolomnamen wordt weergegeven.

In dit menu kun je de kolomnamen selecteren en de pijlen gebruiken om de gewenste kolommen toe te voegen en te verwijderen.

![Dashboard Column Settings](https://lh7-us.googleusercontent.com/cXnnrIR-y4TRDnRE9irGvvjnmkN-HSGEQTh7FiwsjRHzXF7FNjd-_gLO-m55fLlv6lVjk-VvThgdW5JWgqIVZSm5tfk3hC7xrj68uRE5OgIPMtYIrpxOhhYzk4OMibyDBqvHQ0VZaDAysZohlH8dxm8)

<div data-full-width="true"><figure><img src="../../../.gitbook/assets/docbits_dashboard_columns_drag.png" alt="Docbits Dashboard Columns Drag"><figcaption></figcaption></figure></div>

Je kunt de volgorde van de kolommen instellen door op de stippen naast de kolomnaam te klikken en ze naar de juiste positie te slepen.

Een andere optie stelt je ook in staat om velden van een documenttype toe te voegen:

<figure><img src="../../../.gitbook/assets/docbits_dashboard_columns_add_field.png" alt="Docbits Dashboard Columns Add Field" width="243"><figcaption></figcaption></figure>

<figure><img src="../../../.gitbook/assets/docbits_dashboard_columns_select_doctype.png" alt="Docbits Dashboard Columns Select Doctype" width="563"><figcaption><p>Select document type field window</p></figcaption></figure>

Hier kun je kiezen uit de verschillende documenttypes:

<figure><img src="../../../.gitbook/assets/docbits_dashboard_columns_select_doctype_2.png" alt="Docbits Dashboard Columns Select Doctype 2" width="254"><figcaption></figcaption></figure>

Voor elk documenttype zijn er verschillende velden die je kunt toevoegen:

<figure><img src="../../../.gitbook/assets/docbits_dashboard_columns_add_visible.png" alt="Docbits Dashboard Columns Add Visible" width="518"><figcaption><p>Select document type field window</p></figcaption></figure>

Selecteer de velden die je nodig hebt en klik op "Toevoegen aan zichtbare kolommen", ze zullen dan verschijnen in de kolominstellingen.

<figure><img src="../../../.gitbook/assets/image (76).png" alt="" width="281"><figcaption></figcaption></figure>
