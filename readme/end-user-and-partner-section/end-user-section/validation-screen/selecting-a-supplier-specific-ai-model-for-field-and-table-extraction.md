# Selección de un modelo de IA específico de proveedor para extracción de campos y tablas

En la sección de herramientas de la **Pantalla de Validación**, puede elegir el modelo de IA utilizado para la **Extracción de Campos** y la **Extracción de Tablas**.\
El **modelo de IA** seleccionado se aplica solo a todos los documentos del proveedor actual.

Para cambiar el modelo, abra el menú de opciones haciendo clic en los **tres puntos** en el lado derecho de la pantalla, luego elija uno de los modelos disponibles:<br>

* **Full** – Cuesta 2 tokens
* **Fast** – Cuesta 1 token
* **Turbo** – Cuesta 1 token

Si los resultados de la **extracción de campos** o la **extracción de tablas** no son satisfactorios, intente cambiar a otro modelo para mejorar la precisión.

Para configurar el modelo de IA **globalmente para proveedores sin un modelo personalizado** o para **administrar qué modelo está activo para cada proveedor**, consulte la documentación correspondiente [**aquí**](../../../administration-and-setup/settings/document-processing/classification-and-extraction/ai-model.md).
