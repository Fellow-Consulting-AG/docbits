# Registro de Fornecedor

1.  Você receberá um convite com um link por E-mail para iniciar o processo de registro. Por favor, clique em “Ir para Registro”, envie as informações necessárias e aceite os termos e condições para começar. \


    <div align="left"><figure><img src="../../../.gitbook/assets/supplier_registration_1.png" alt="" width="287"><figcaption></figcaption></figure> <figure><img src="../../../.gitbook/assets/supplier_registration_2.png" alt="" width="246"><figcaption></figcaption></figure></div>
2.  **Detalhes do Fornecedor**

    Por favor, verifique se o nome do Fornecedor está correto. Este é um detalhe crítico para fins de identidade e legais que não pode ser alterado. Por favor, forneça todas as informações aplicáveis, incluindo os campos obrigatórios. Quando concluído, clique em “Próximo” para continuar para a seção de Dados de Endereço. Para salvar seu progresso e continuar depois, clique em “Salvar.”\


    <figure><img src="../../../.gitbook/assets/supplier_registration_3.png" alt="" width="563"><figcaption></figcaption></figure>
3.  **Dados de Endereço** Por favor, forneça todas as informações aplicáveis, incluindo os campos obrigatórios. Quando concluído, clique em “Próximo” para continuar para a seção de Contato do Fornecedor. Para salvar seu progresso e continuar depois, clique em “Salvar.”\


    <figure><img src="../../../.gitbook/assets/supplier_registration_4.png" alt="" width="563"><figcaption></figcaption></figure>
4.  **Contato do Fornecedor** Por favor, forneça todas as informações aplicáveis, incluindo os campos obrigatórios. Quando concluído, clique em “Próximo” para continuar para a seção Financeira do Fornecedor. Para salvar seu progresso e continuar depois, clique em “Salvar.”\


    <figure><img src="../../../.gitbook/assets/supplier_registration_5.png" alt="" width="404"><figcaption></figcaption></figure>
5.  **Financeiro do Fornecedor** Por favor, forneça todas as informações aplicáveis. Quando concluído, clique em “Próximo” para continuar para a seção de Compras do Fornecedor. Para salvar seu progresso e continuar depois, clique em “Salvar.”\


    <figure><img src="../../../.gitbook/assets/supplier_registration_6.png" alt="" width="402"><figcaption></figcaption></figure>
6.  **Compras do Fornecedor** Por favor, forneça todas as informações aplicáveis. Quando concluído, clique em “Próximo” para continuar para a seção de Certificados do Fornecedor. Para salvar seu progresso e continuar depois, clique em “Salvar.”\


    <figure><img src="../../../.gitbook/assets/supplier_registration_7.png" alt="" width="401"><figcaption></figcaption></figure>
7.  **Certificados do Fornecedor** Por favor, faça o upload de todos os certificados necessários (Imposto, Seguro, Diversificação de Fornecedores). Por favor, salve seu progresso se precisar retornar para completar o processo de registro mais tarde. Quando concluído, clique em “Enviar para Aprovação.”\


    <figure><img src="../../../.gitbook/assets/supplier_registration_8.png" alt="" width="404"><figcaption></figcaption></figure>
8.  Por favor, faça o upload de todos os certificados necessários (Imposto, Seguro, Diversificação de Fornecedores). Por favor, salve seu progresso se precisar retornar para completar o processo de registro mais tarde. Quando concluído, clique em “Enviar para Aprovação.”\


    <figure><img src="../../../.gitbook/assets/supplier_registration_9.png" alt="" width="404"><figcaption></figcaption></figure>
