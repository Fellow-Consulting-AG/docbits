# General Information

### What types of documents does DocBits support?

* Invoice&#x20;
* Credit Note&#x20;
* Delivery Note
* Order Confirmation
* Purchase Order&#x20;
* Custom Documents

### Does DocBits support custom document types ?&#x20;

* Yes Users are able to create there own document types.

### What file types does DocBits support ?

* pdf
* .edi
* .xml
* .tiff  (.tif)
* .jpeg  (.jpg)
* .png
