# General Information

<details>

<summary>What types of documents does DocBits support?</summary>

#### Supported document types

* Invoice&#x20;

- Credit Note&#x20;

* Delivery Note

- Order Confirmation

* Purchase Order&#x20;

- Custom Documents

</details>



<details>

<summary>Does DocBits support custom document types ? </summary>

* Yes Users are able to create there own document types.

</details>

* .jpeg  (.jpg)
