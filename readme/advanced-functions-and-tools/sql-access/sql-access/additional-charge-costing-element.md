# Extra Kosten Element

**Tabelnaam:** `additional_charge_costing_element`

Deze tabel slaat informatie op over extra kostenelementen die aan documenten kunnen worden gekoppeld.

| Kolomnaam | Gegevenstype | Beschrijving |
| :--- | :--- | :--- |
| `id` | `String` | Unieke identifier voor het extra kostenelement. |
| `name` | `String` | De naam van het extra kostenelement. |
| `created_at` | `DateTime` | Tijdstempel wanneer het record is aangemaakt. |
| `updated_at` | `DateTime` | Tijdstempel wanneer het record voor het laatst is bijgewerkt. |
| `organization_id` | `String` | Identifier voor de organisatie waartoe dit element behoort. |
