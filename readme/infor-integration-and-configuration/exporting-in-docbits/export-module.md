# Export Module

## Overzicht

De Export-pagina toont alle geconfigureerde exportinstellingen, inclusief of ze actief of inactief zijn. Vanaf hier kunnen gebruikers:

* Bekijk en beheer bestaande exportconfiguraties
* Maak nieuwe exportverbindingen aan (bijv. naar **Infor**, **Infor & IDM**, **Webhook** of **SFTP**)
* Bewerk of verwijder bestaande exportconfiguraties

## Waar te vinden

Je kunt het vinden onder: **Instellingen** → **Documentverwerking** → **Export**
