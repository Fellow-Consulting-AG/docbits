# WatchDog Konfiguration

Diese Anleitung erklärt, wie Sie WatchDog innerhalb der DocBits-Anwendung konfigurieren.

## 1. Übersicht

Rufen Sie die WatchDog-Einstellungen über das DocBits-Einstellungsmenü auf.

<figure><img src="../../.gitbook/assets/watchdog_v2_01.png" alt="WatchDog Übersicht"><figcaption></figcaption></figure>

## 2. Ordner-Einstellungen

Konfigurieren Sie die Eingabe- und Ausgabeverzeichnisse für WatchDog. Stellen Sie sicher, dass diese Pfade mit den während der Installation erstellten Verzeichnissen übereinstimmen.

<figure><img src="../../.gitbook/assets/watchdog_v2_02.png" alt="WatchDog Ordner-Einstellungen"><figcaption></figcaption></figure>

## 3. Allgemeine Einstellungen

Konfigurieren Sie allgemeine Einstellungen wie Dokumenttypen und andere Verarbeitungsoptionen.

<figure><img src="../../.gitbook/assets/watchdog_v2_03.png" alt="WatchDog Allgemeine Einstellungen"><figcaption></figcaption></figure>

## 4. Export-Einstellungen

Richten Sie die Exportziele und -methoden ein.

<figure><img src="../../.gitbook/assets/watchdog_v2_04.png" alt="WatchDog Export-Einstellungen"><figcaption></figcaption></figure>

## 5. Export-Konfigurationsdetails

Zeigen Sie spezifische Exportkonfigurationen an und verwalten Sie diese.

<figure><img src="../../.gitbook/assets/watchdog_v2_05.png" alt="WatchDog Export-Konfigurationsdetails"><figcaption></figcaption></figure>

## 6. BOD-Mapping-Konfiguration

Konfigurieren Sie das BOD-Mapping für die Integration mit Infor.

<figure><img src="../../.gitbook/assets/watchdog_v2_06.png" alt="WatchDog BOD-Mapping"><figcaption></figcaption></figure>

## 7. Konfiguration herunterladen

Nach der Konfiguration laden Sie die Konfigurationsdatei herunter, um sie mit der WatchDog-Anwendung zu verwenden.

<figure><img src="../../.gitbook/assets/watchdog_v2_07.png" alt="WatchDog Konfiguration herunterladen"><figcaption></figcaption></figure>
