# V1

## Vereisten

| Vereiste informatie | Beschrijving |
| ------------------- | ----------- |
| Logingegevens voor Cloud | Inloggegevens zijn verplicht voor toegang tot de Infor Cloud-omgeving. De gebruiker moet de rollen "Infor-SystemAdministrator" and "UserAdmin" hebben. |

* Ga naar INSTELLINGEN (in de bovenste balk) and selecteer INTEGRATIE. Onder Instellingen SSO-serviceprovider vindt u alle informatie die u nodig heeft voor de volgende stappen.
* Download het certificaat.

## De SAML-METAGEGEVENS exporteren.

Het bestand ziet er als volgt uit: ServiceProviderSAMLMetadata\_10\_20\_2021.xml

Importeer de SAML-METAGEGEVENS in de SSO-instellingen.

Ga naar de INSTELIINGEN IDENTITEITSSERVICEPROVIDER, die u vindt onder INTEGRATIES in de INSTELLINGEN. Voer uw Tenant ID in (bijv. FELLOWPRO\_DEV) and daaronder ziet u de regel Bestand uploaden and de IMPORT-knop, waar u het eerder geÃ«xporteerde SAML-metagegevensbestand moet uploaden.

* Klik op IMPORT and kies vervolgens het METADATA-bestand dat u al heeft gedownload uit de INSTELLINGEN SSO-SERVICEPROVIDER.
* Klik op CONFIGURE.
