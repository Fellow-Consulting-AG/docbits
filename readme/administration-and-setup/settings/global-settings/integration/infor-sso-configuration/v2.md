# V2

## Vereisten

| Vereiste informatie | Beschrijving |
| ------------------- | ----------- |
| Logingegevens voor Cloud | Inloggegevens zijn verplicht voor toegang tot de Infor Cloud-omgeving. De gebruiker moet de rollen "Infor-SystemAdministrator" and "UserAdmin" hebben. |
| Config Admin-gegevens (DocBits) | U zou een e-mail van FellowPro AG moeten hebben ontvangen met de inloggegevens voor de DocBits SSO-instellingenpagina. U heeft een gebruikersnaam and wachtwoord nodig. |
| Certificaat | U kunt het certificaat downloaden in DocBits onder Instellingen SSO-serviceprovider |

## Ga naar de INSTELLINGEN SSO-SERVICEPROVIDER in DocBits

* Log in op URL https://app.docbits.com/ met de inloggegevens die u van ons heeft ontvangen.
* Ga naar INSTELLINGEN (in de bovenste balk) and selecteer INTEGRATIE. Onder Instellingen SSO-serviceprovider vindt u alle informatie die u nodig heeft voor de volgende stappen.
* Download het certificaat.

## De SAML-METAGEGEVENS exporteren.

Het bestand ziet er als volgt uit: ServiceProviderSAMLMetadata_10_20_2021.xml

Importeer de SAML-METAGEGEVENS in de SSO-instellingen.

Ga naar de INSTELIINGEN IDENTITEITSSERVICEPROVIDER, die u vindt onder INTEGRATIES in de INSTELLINGEN. Voer uw Tenant ID in (bijv. FELLOWPRO_DEV) and daaronder ziet u de regel Bestand uploaden and de IMPORT-knop, waar u het eerder geÃ«xporteerde SAML-metagegevensbestand moet uploaden.

* Klik op IMPORT and kies vervolgens het METADATA-bestand dat u al heeft gedownload uit de INSTELLINGEN SSO-SERVICEPROVIDER.
* Klik op CONFIGURE.

![](https://lh7-us.googleusercontent.com/7-v_YNgl_29WrK2lE62nEfIRQ3R5KVmOL_PeR8_ZxS8LNxHSVpHuKcNwDAmaSGTNepi0Izg64T3l3FY6XUSMZCVB-kyV3cbf0DtI-9GnspkrSibmRW3Dx2ESxZeyrkseRYRKdnmUn-GR4fmh8gUx_Rg)
