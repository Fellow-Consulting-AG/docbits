# Erstellen eines neuen Untertyps

## Hier ist eine Schritt-für-Schritt-Anleitung zum Erstellen eines neuen Untertyps:

*   **Navigieren Sie zum Einstellungsbereich:** Melden Sie sich als Administrator bei DocBits an und navigieren Sie zum Bereich Dokumenttypenverwaltung.

<figure><img src="../../../../.gitbook/assets/Bildschirmfoto 2024-05-21 um 09.02.56.png" alt=""><figcaption></figcaption></figure>

*   **Wählen Sie die Option zum Hinzufügen eines Untertyps:** Klicken Sie auf die Schaltfläche „+ Neu“, um einen neuen Untertyp hinzuzufügen.

<figure><img src="../../../../.gitbook/assets/Bildschirmfoto 2024-05-21 um 09.04.46.png" alt=""><figcaption></figcaption></figure>

*   **Benennen Sie den Untertyp:** Geben Sie einen beschreibenden Namen für den neuen Untertyp ein. Dieser Name sollte den Zweck des Untertyps klar beschreiben, damit Benutzer leicht verstehen können, welche Art von Dokumenten er darstellt.

<figure><img src="https://lh7-us.googleusercontent.com/hFxoJh9mFR52IbFtRGTstXUGjUc2u9fBzlya5kPj18REi9ampOKfek6NPaNje_fGash3brlAZkEw8MHqDPpNEPCvnb122xYiHMqwVzV35FQdoev8XLXiu7DogUuGjsoun51sum8Z8PLZo2xw9bvImRc" alt="" width="375"><figcaption></figcaption></figure>

<figure><img src="https://lh7-us.googleusercontent.com/EjBS5qKWAm0SoZhLkhT2KW1q5F89vZ6FCLZ2ZzvxfJPmVE9DyyAUXieHA_56guDbKsuEvNDY9DyA-uUMx8Q1UDMSohmhyIVnnLfZaPiXrVSGsXpnjEgtpF7GtOOiRX8VsXBpu2-3lpqc0zu2LdzIYZk" alt=""><figcaption></figcaption></figure>

*   **Initiale Einstellungen konfigurieren:** Legen Sie die initialen Einstellungen für den neuen Untertyp fest, einschließlich der Standardfelder, Optionen und Vorlagen, die für diesen Untertyp verwendet werden sollen. Dies kann das Hinzufügen spezifischer Metadatenfelder, das Festlegen von Genehmigungsworkflows oder das Konfigurieren von Benutzerberechtigungen umfassen.
*   **Optionale Konfigurationen vornehmen:** Abhängig von den Anforderungen Ihres Unternehmens oder der Art der Dokumente können Sie zusätzliche Konfigurationen vornehmen, um den neuen Untertyp an Ihre spezifischen Bedürfnisse anzupassen. Dies kann das Festlegen von Standardwerten, Validierungsregeln oder benutzerdefinierten Aktionen umfassen.

<figure><img src="https://lh7-us.googleusercontent.com/d3-6iNTABnLH9KbWlM1NMu8Nsy2-KbqTd2elHgZYF1jwZOKqP17qttUCD4j676VJN3t7J0I85wHjr8TIfv3UGgLVxNvV4ns2I82EfCKHbyfIAUA8ML6IbLzoLH4Ucn57XzgUNZLO9LeCLzlk_m-M3EQ" alt=""><figcaption><p>Aktion</p></figcaption></figure>

*   **Speichern Sie den neuen Untertyp:** Sobald Sie alle erforderlichen Informationen eingegeben haben, speichern Sie den neuen Untertyp, um ihn im Dokumentenmanagementsystem zu erstellen.

Nachdem der neue Untertyp erstellt wurde, können Benutzer Dokumente dieses Typs gemäß den von Ihnen festgelegten initialen Einstellungen hinzufügen und verwalten. Stellen Sie sicher, dass Sie die Benutzer über den neuen Untertyp informieren und bei Bedarf Schulungen oder Anleitungen bereitstellen, damit sie ihn effektiv nutzen können.
