# EDI Videos

Diese Seite enthält eine Reihe kurzer Videoanleitungen, die erklären, wie DocBits EDI-Dokumente verarbeitet. Die folgenden Videos veranschaulichen verschiedene Aspekte des Prozesses, einschließlich eines Beispiels mit einer EDI 850 (Bestellung).

## **Verstehen der Strukturbeschreibungsdatei**

Dieses Video erklärt den Zweck der Strukturbeschreibungsdatei, die von DocBits verwendet wird, um EDI-Dokumente zu interpretieren.

{% embed url="https://youtu.be/_U6yyYFMo84" %}

**Verwandter Leitfaden**:
Für eine detaillierte Anleitung und Bearbeitungsanweisungen siehe den [**Leitfaden zur Strukturbeschreibungsdatei**](edi-structure-descriptor-file-guide/)

**Verwandter Leitfaden**:
