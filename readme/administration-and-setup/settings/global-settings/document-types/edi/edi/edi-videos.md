# Videos EDI

Esta página contiene una serie de breves guías en video que explican cómo DocBits procesa los documentos EDI. Los siguientes videos ilustran diferentes aspectos del proceso, incluido un ejemplo utilizando una EDI 850 (Orden de Compra).

## **Comprendiendo el Archivo Descriptor de Estructura**

&#x20;Este video explica el propósito del archivo descriptor de estructura utilizado por DocBits para interpretar documentos EDI.

**Guía Relacionada**:
Para un recorrido detallado e instrucciones de edición, consulte la [**Guía del Archivo Descriptor de Estructura**](edi-structure-descriptor-file-guide/)

**Guía Relacionada**:
