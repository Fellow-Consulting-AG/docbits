# Mapeamento XRechnung no DocBits

No DocBits, as faturas XRechnung são mapeadas para campos específicos usando uma configuração predefinida que garante que os dados possam ser exportados perfeitamente para vários formatos, incluindo a integração com outros sistemas como o **Infor**. A configuração de exportação aproveita modelos e regras para garantir que cada elemento da XRechnung seja capturado e mapeado adequadamente.

### Componentes principais

1.  **Tipos de Documento**: Os documentos XRechnung são mapeados para **Tipos de Documento** específicos no DocBits. Cada tipo de documento (por exemplo, fatura, nota de crédito, nota de débito) tem sua própria estrutura e campos.

2.  **Mapeamento de Campos**: Os campos na XRechnung são mapeados para os campos correspondentes no DocBits usando um **arquivo de configuração de exportação**. Este arquivo define como cada campo XRechnung é manipulado e para onde deve ser exportado.

3.  **Regras para Exportação**: Algumas regras são definidas para lidar com casos específicos onde os valores podem diferir, incluindo verificações de tolerância, requisitos de aprovação ou encargos no nível da linha. Essas regras garantem que os dados da XRechnung sejam processados e exportados corretamente, com base em uma lógica de negócios específica.
