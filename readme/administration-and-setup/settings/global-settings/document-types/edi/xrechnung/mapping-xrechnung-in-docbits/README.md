# Mappatura di XRechnung in DocBits

In DocBits, le fatture XRechnung sono mappate su campi specifici utilizzando una configurazione predefinita che garantisce che i dati possano essere esportati senza problemi in vari formati, inclusa l'integrazione con altri sistemi come **Infor**. La configurazione di esportazione sfrutta modelli e regole per garantire che ogni elemento della XRechnung venga acquisito e mappato in modo appropriato.

### Componenti chiave

1.  **Tipi di documento**: i documenti XRechnung sono mappati su **Tipi di documento** specifici in DocBits. Ogni tipo di documento (es. fattura, nota di credito, nota di debito) ha la propria struttura e i propri campi.

2.  **Mappatura dei campi**: i campi nella XRechnung sono mappati sui campi corrispondenti in DocBits utilizzando un **file di configurazione dell'esportazione**. Questo file definisce come viene gestito ogni campo XRechnung e dove deve essere esportato.

3.  **Regole per l'esportazione**: vengono definite alcune regole per gestire casi specifici in cui i valori possono differire, inclusi controlli di tolleranza, requisiti di approvazione o oneri a livello di riga. Queste regole garantiscono che i dati XRechnung vengano elaborati ed esportati correttamente, in base a una specifica logica aziendale.
