# Mapeo de XRechnung en DocBits

En DocBits, las facturas XRechnung se mapean a campos específicos utilizando una configuración predefinida que garantiza que los datos se puedan exportar sin problemas a varios formatos, incluida la integración con otros sistemas como **Infor**. La configuración de exportación aprovecha plantillas y reglas para garantizar que cada elemento de XRechnung se capture y mapee de manera adecuada.

### Componentes clave

1.  **Tipos de documento**: los documentos XRechnung se mapean a **Tipos de documento** específicos en DocBits. Cada tipo de documento (p. ej., factura, nota de crédito, nota de débito) tiene su propia estructura y campos.

2.  **Mapeo de campos**: los campos en XRechnung se mapean a los campos correspondientes en DocBits utilizando un **archivo de configuración de exportación**. Este archivo define cómo se maneja cada campo de XRechnung y adónde debe exportarse.

3.  **Reglas para la exportación**: se definen ciertas reglas para manejar casos específicos donde los valores pueden diferir, incluidos controles de tolerancia, requisitos de aprobación o cargos a nivel de línea. Estas reglas garantizan que los datos de XRechnung se procesen y exporten correctamente, basándose en una lógica de negocio específica.
