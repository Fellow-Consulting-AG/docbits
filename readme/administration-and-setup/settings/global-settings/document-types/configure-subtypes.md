# Configurar subtipos

La configuración de subtipos le permite personalizar la estructura y el comportamiento de los documentos dentro de un tipo específico. Aquí hay una explicación de cómo puede usar las opciones "Campos", "Editar diseño" y "Scripts" para personalizar cada subtipo a necesidades específicas:

*   **Campos:** La opción "Campos" le permite agregar, editar o eliminar campos de metadatos personalizados para el subtipo. Estos campos pueden contener información sobre los documentos de ese tipo, como título, autor, fecha, categoría, etc. Puede usar diferentes tipos de campos como cuadros de texto, listas desplegables, valores de fecha, etc., para capturar los datos según sus requisitos.

*   **Editar diseño:** La opción "Editar diseño" le permite personalizar la apariencia y disposición de los campos en la interfaz de usuario. Puede cambiar el orden de los campos, crear grupos de campos para agrupar información relacionada y ajustar el tamaño y la posición de los campos en la página. Esto le permite optimizar la experiencia del usuario y mejorar la usabilidad.

<figure><img src="https://lh7-us.googleusercontent.com/thrOOaF54hZqhl1XVvgZfT1Hu4NOS1550gA9EmksjGaKTMLfKCEQct-OIM8e2qDh0E9Y9K8yf9CJ8kv1jmxgWSbqnAnk9dJpqA-fEhhU2scaMiR-xkcK_UFfiaSffvXb7e1Uvt-_y7U6fr9gAccJtfM" alt=""><figcaption></figcaption></figure>

*   **Scripts:** La opción "Scripts" le permite agregar lógica personalizada o automatización para el subtipo. Puede usar scripts para activar acciones específicas cuando se crea, edita o elimina un documento de este tipo. Esto puede ser útil para implementar reglas comerciales complejas, realizar validaciones o integrar sistemas externos.
