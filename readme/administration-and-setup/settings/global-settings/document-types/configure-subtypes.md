# Configurer les sous-types

La configuration des sous-types vous permet de personnaliser la structure et le comportement des documents au sein d'un type spécifique. Voici une explication de la façon dont vous pouvez utiliser les options "Champs", "Modifier la mise en page" et "Scripts" pour personnaliser chaque sous-type selon des besoins spécifiques :

*   **Champs :** L'option "Champs" vous permet d'ajouter, de modifier ou de supprimer des champs de métadonnées personnalisés pour le sous-type. Ces champs peuvent contenir des informations sur les documents de ce type tels que le titre, l'auteur, la date, la catégorie, etc. Vous pouvez utiliser différents types de champs tels que des zones de texte, des listes déroulantes, des valeurs de date, etc. pour capturer les données selon vos besoins.

*   **Modifier la mise en page :** L'option "Modifier la mise en page" vous permet de personnaliser l'apparence et la disposition des champs sur l'interface utilisateur. Vous pouvez modifier l'ordre des champs, créer des groupes de champs pour regrouper des informations connexes, et ajuster la taille et la position des champs sur la page. Cela vous permet d'optimiser l'expérience utilisateur et d'améliorer la convivialité.

<figure><img src="https://lh7-us.googleusercontent.com/thrOOaF54hZqhl1XVvgZfT1Hu4NOS1550gA9EmksjGaKTMLfKCEQct-OIM8e2qDh0E9Y9K8yf9CJ8kv1jmxgWSbqnAnk9dJpqA-fEhhU2scaMiR-xkcK_UFfiaSffvXb7e1Uvt-_y7U6fr9gAccJtfM" alt=""><figcaption></figcaption></figure>

*   **Scripts :** L'option "Scripts" vous permet d'ajouter une logique personnalisée ou une automatisation pour le sous-type. Vous pouvez utiliser des scripts pour déclencher des actions spécifiques lorsqu'un document de ce type est créé, modifié ou supprimé. Cela peut être utile pour mettre en œuvre des règles métier complexes, effectuer des validations ou intégrer des systèmes externes.
