# Trainingsdaten verwalten

## Verwalten vorhandener Trainingsdaten, einschließlich Hinzufügen, Bearbeiten oder Entfernen von Datensätzen.



**Um Trainingsdaten effektiv zu verwalten, können Sie folgende Schritte unternehmen:**&#x20;

**Hinzufügen neuer Datensätze:**&#x20;

* Sammeln Sie neue Belege, die als Trainingsdaten für Ihr Modell dienen sollen.
* Stellen Sie sicher, dass diese Belege eine repräsentative Stichprobe der verschiedenen Datentypen sind, die das Modell verarbeiten soll.
* Laden Sie die neuen Datensätze in Ihr Trainingsdaten-Repository hoch.

**Bearbeiten vorhandener Datensätze:**&#x20;

* Überprüfen Sie regelmäßig Ihre vorhandenen Trainingsdaten und aktualisieren Sie diese nach Bedarf. Dies kann das Bearbeiten von Beleg-Metadaten, das Hinzufügen zusätzlicher Labels oder das Entfernen fehlerhafter oder nicht repräsentativer Datensätze umfassen.

**Entfernen von Datensätzen:**&#x20;

* Identifizieren Sie veraltete, ungenaue oder nicht mehr relevante Datensätze und entfernen Sie diese aus Ihrem Trainingsdatensatz.
* Stellen Sie sicher, dass Sie einen klaren Prozess für die Entscheidung haben, welche Datensätze entfernt werden sollen, und dokumentieren Sie diesen Prozess.

**Versionierung von Trainingsdaten:**&#x20;

* Implementieren Sie ein Versionskontrollsystem für Ihre Trainingsdaten, um Änderungen zu verfolgen und einen klaren Verlauf der Datensatzänderungen beizubehalten. Dies ermöglicht es Ihnen, bei Bedarf ältere Versionen der Trainingsdaten wiederherzustellen und Änderungen nachzuvollziehen.

**Sicherheit der Trainingsdaten:**&#x20;

* Stellen Sie sicher, dass Ihre Trainingsdaten angemessen geschützt sind, insbesondere wenn sie sensible oder vertrauliche Informationen enthalten. Implementieren Sie Zugriffskontrollen, um sicherzustellen, dass nur autorisierte Benutzer auf die Trainingsdaten zugreifen können, und verschlüsseln Sie die Daten während der Übertragung und Speicherung.

**Dokumentation und Verfolgung:**&#x20;

* Dokumentieren Sie alle Änderungen an Ihren Trainingsdaten, einschließlich des Hinzufügens, Bearbeitens und Entfernens von Datensätzen. Dies ermöglicht es Ihnen, die Historie Ihrer Trainingsdaten zu verfolgen und sicherzustellen, dass Sie über aktuelle und relevante Daten für das Training Ihres Modells verfügen.

Durch die regelmäßige Verwaltung und Aktualisierung Ihrer Trainingsdaten können Sie sicherstellen, dass Ihr Modell mit aktuellen und repräsentativen Daten trainiert wird und eine optimale Leistung erzielt.
