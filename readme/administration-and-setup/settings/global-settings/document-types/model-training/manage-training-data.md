# Gestionar datos de entrenamiento

## Gestione los datos de entrenamiento existentes, incluida la adición, edición o eliminación de registros.



**Para gestionar eficazmente los datos de entrenamiento, puede seguir estos pasos:**

**Agregar nuevos registros:**

* Recopile nuevos documentos para que sirvan como datos de entrenamiento para su modelo.
* Asegúrese de que estos documentos sean una muestra representativa de los diferentes tipos de datos que el modelo está diseñado para procesar.
* Cargue los nuevos registros en su repositorio de datos de entrenamiento.

**Editar registros existentes:**

* Revise periódicamente sus datos de entrenamiento existentes y actualícelos según sea necesario. Esto puede incluir la edición de metadatos de documentos, la adición de etiquetas adicionales o la eliminación de registros erróneos o no representativos.

**Eliminar registros:**

* Identifique registros obsoletos, inexactos o que ya no sean relevantes y elimínelos de su conjunto de datos de entrenamiento.
* Asegúrese de tener un proceso claro para decidir qué registros eliminar y documente ese proceso.

**Control de versiones de datos de entrenamiento:**

* Implemente un sistema de control de versiones para sus datos de entrenamiento para rastrear los cambios y mantener un historial claro de los cambios en el conjunto de datos. Esto le permite restaurar versiones anteriores de los datos de entrenamiento cuando sea necesario y realizar un seguimiento de los cambios.

**Seguridad de los datos de entrenamiento:**

* Asegúrese de que sus datos de entrenamiento estén protegidos adecuadamente, especialmente si contienen información confidencial o sensible. Implemente controles de acceso para garantizar que solo los usuarios autorizados puedan acceder a los datos de entrenamiento, y cifre los datos durante la transferencia y el almacenamiento.

**Documentación y seguimiento:**

* Documente todos los cambios en sus datos de entrenamiento, incluida la adición, edición y eliminación de conjuntos de datos. Esto le permite rastrear el historial de sus datos de entrenamiento y asegurarse de tener datos actuales y relevantes para entrenar su modelo.

Al gestionar y actualizar periódicamente sus datos de entrenamiento, puede asegurarse de que su modelo se entrene con datos actuales y representativos y logre un rendimiento óptimo.
