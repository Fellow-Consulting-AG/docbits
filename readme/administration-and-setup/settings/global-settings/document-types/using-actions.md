# Aktionen verwenden

Hier ist eine Anleitung, wie Sie die Aktionen "Kopieren" und "Löschen" für eine effiziente Subtyp-Verwaltung richtig verwenden:

## **Kopieren eines Subtyps:**

* Navigieren Sie zu den Subtyp-Verwaltungseinstellungen in Ihrem Dokumentenmanagementsystem.
* Wählen Sie den Subtyp aus, den Sie kopieren möchten, klicken Sie auf "Kopieren" und geben Sie bei Bedarf einen neuen Namen für den kopierten Subtyp ein.
* Bestätigen Sie die Aktion, und das System erstellt eine Kopie des ausgewählten Subtyps mit allen vorhandenen Einstellungen, Feldern, Layouts und Skripten.

## Löschen eines Subtyps:

* Navigieren Sie zu den Subtyp-Verwaltungseinstellungen und wählen Sie den Subtyp aus, den Sie löschen möchten.
* Klicken Sie auf das Mülleimer-Symbol rechts im Aktionsmenü.
* Bestätigen Sie die Löschaktion, indem Sie eine Bestätigungsnachricht akzeptieren, falls Sie dazu aufgefordert werden.
* Beachten Sie, dass das Löschen eines Subtyps unwiderruflich alle damit verbundenen Dokumente und Daten entfernen kann. Stellen Sie sicher, dass Sie alle notwendigen Sicherheitsvorkehrungen treffen und prüfen Sie, ob der Subtyp nicht mehr benötigt wird, bevor Sie ihn löschen.

Durch die ordnungsgemäße Verwendung dieser Aktionen können Sie die Verwaltung von Subtypen optimieren. Das Kopieren ermöglicht es Ihnen, vorhandene Konfigurationen für neue Subtypen zu nutzen, während das Löschen eine effiziente Bereinigung nicht mehr benötigter Subtypen ermöglicht. Es ist jedoch wichtig, beim Löschen vorsichtig zu sein, um Datenverlust zu vermeiden.
