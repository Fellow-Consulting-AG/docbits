# Verwendung der Vorlagenvorschau

## Die Vorlagenvorschau ist ein äußerst nützliches Werkzeug, um Layouteinstellungen in Echtzeit zu überprüfen und anzupassen, bevor Änderungen an einem Beleg oder einer Vorlage vorgenommen werden.

### Hier ist eine Erklärung, wie Sie die Vorlagenvorschau verwenden können:

<figure><img src="../../../../../.gitbook/assets/image (152).png" alt=""><figcaption></figcaption></figure>

#### Öffnen der Vorlagenvorschau:

* Normalerweise finden Sie die Option "Vorlagenvorschau" in der Benutzeroberfläche des Vorlagen-Editors.

<figure><img src="../../../../../.gitbook/assets/image (153).png" alt=""><figcaption></figcaption></figure>

#### Auswahl der Vorlage:

* Wählen Sie die Vorlage aus, deren Layout Sie überprüfen möchten.
* Dies kann eine bestehende Vorlage sein, an der Sie Änderungen vornehmen möchten, oder eine neue Vorlage, die Sie erstellen möchten.

<figure><img src="../../../../../.gitbook/assets/docbits_template_preview_adjust.png" alt="Docbits Template Preview Adjust" width="186"><figcaption></figcaption></figure>

#### Anpassen der Layouteinstellungen:

* Ändern Sie die Layouteinstellungen nach Bedarf. Dies kann das Hinzufügen, Entfernen oder Anpassen von Gruppen, Feldern, Spalten, Zeilen, Schriftarten usw. beinhalten.

<figure><img src="../../../../../.gitbook/assets/image (155).png" alt="" width="271"><figcaption></figcaption></figure>

#### Echtzeit-Vorschau:

* Wenn Sie die Layouteinstellungen ändern, wird die Vorschau in Echtzeit aktualisiert.
* Sie können sofort sehen, wie sich Ihre Änderungen auf das Aussehen und die Struktur der Vorlage auswirken.

#### Interaktive Anpassung:

* Nutzen Sie die Möglichkeit, das Layout interaktiv anzupassen, indem Sie Elemente verschieben, in der Größe ändern oder andere Anpassungen vornehmen, während Sie die Auswirkungen in Echtzeit in der Vorschau überprüfen.

#### Testen verschiedener Konfigurationen:

* Experimentieren Sie mit verschiedenen Layout-Konfigurationen, um das beste Design für Ihre Anforderungen zu finden.
* Verwenden Sie die Vorlagenvorschau, um zu sehen, wie jede Änderung das endgültige Aussehen beeinflusst.

#### Änderungen speichern und übernehmen:

* Sobald Sie mit dem Layout zufrieden sind, speichern Sie Ihre Änderungen.
* Je nach Software finden Sie möglicherweise auch die Option, Ihre Änderungen direkt zu übernehmen, um die Vorlage für die Verwendung in anderen Belegen oder Prozessen zu aktualisieren.

Die Verwendung der Vorlagenvorschau ermöglicht es Ihnen, sicherzustellen, dass Ihr Layout Ihren gewünschten Anforderungen entspricht, bevor Sie Änderungen endgültig übernehmen. Dies ermöglicht es Ihnen, das Design und die Struktur Ihrer Belege effizient anzupassen und sicherzustellen, dass sie den gewünschten visuellen und funktionalen Standards entsprechen.
