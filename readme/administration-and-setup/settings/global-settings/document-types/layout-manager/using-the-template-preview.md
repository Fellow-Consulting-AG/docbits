# Korzystanie z podglądu szablonu

## Podgląd szablonu to niezwykle przydatne narzędzie do sprawdzania i dostosowywania ustawień układu w czasie rzeczywistym przed wprowadzeniem zmian w dokumencie lub szablonie.

### Oto wyjaśnienie, jak korzystać z podglądu szablonu:

<figure><img src="../../../../../.gitbook/assets/docbits_script_python_example_2.png" alt="Docbits Script Python Example 2"><figcaption></figcaption></figure>

#### Otwieranie podglądu szablonu:

* Opcję Podgląd szablonu można zazwyczaj znaleźć w interfejsie edytora szablonów.

<figure><img src="../../../../../.gitbook/assets/docbits_script_example_2.png" alt="Docbits Script Example 2"><figcaption></figcaption></figure>

#### Wybór szablonu:

* Wybierz szablon, którego układ chcesz sprawdzić.
* Może to być istniejący szablon, w którym chcesz wprowadzić zmiany, lub nowy szablon, który chcesz utworzyć.

<figure><img src="../../../../../.gitbook/assets/docbits_template_preview_adjust.png" alt="Docbits Template Preview Adjust" width="186"><figcaption></figcaption></figure>

#### Dostosowywanie ustawień układu:

* Zmień ustawienia układu według potrzeb. Może to obejmować dodawanie, usuwanie lub dostosowywanie grup, pól, kolumn, wierszy, czcionek itp.

<figure><img src="../../../../../.gitbook/assets/docbits_script_save_name.png" alt="Docbits Script Save Name" width="271"><figcaption></figcaption></figure>

#### Podgląd w czasie rzeczywistym:

* Podczas zmiany ustawień układu podgląd aktualizuje się w czasie rzeczywistym.
* Możesz natychmiast zobaczyć, jak zmiany wpływają na wygląd i strukturę szablonu.

#### Interaktywna personalizacja:

* Skorzystaj z możliwości interaktywnego dostosowywania układu poprzez przenoszenie, zmianę rozmiaru lub wprowadzanie innych poprawek do elementów, sprawdzając efekty w czasie rzeczywistym w podglądzie.

#### Testowanie różnych konfiguracji:

* Eksperymentuj z różnymi konfiguracjami układu, aby znaleźć najlepszy projekt dla swoich potrzeb.
* Użyj podglądu szablonu, aby zobaczyć, jak każda zmiana wpływa na końcowy wygląd.

#### Zapisywanie i zatwierdzanie zmian:

* Gdy będziesz zadowolony z układu, zapisz zmiany.
* W zależności od oprogramowania może być również dostępna opcja bezpośredniego zatwierdzenia zmian w celu aktualizacji szablonu do użytku w innych dokumentach lub procesach.

Korzystanie z podglądu szablonu pozwala upewnić się, że układ spełnia pożądane potrzeby przed zatwierdzeniem zmian. Pozwala to na wydajne dostosowanie projektu i struktury dokumentów oraz zapewnienie, że spełniają one pożądane standardy wizualne i funkcjonalne.
