# Gestor de Regex

Esta función de DocBits le ofrece una alternativa a la clasificación por modelos, ya que le permite escribir expresiones regulares buscables para un tipo de documento para fines de clasificación y otros.

Tipo de Documento: El Gestor de Regex le permite escribir expresiones regulares y esta regex se buscará en el documento; si encuentra una coincidencia con la regex de un documento definido, entonces clasifica ese documento en el tipo de documento correspondiente. Por ejemplo, si escribió una expresión regular para encontrar “Gutschrift”. Si DocBits encontrara este término en un documento, clasificaría ese documento como una nota de crédito.

Origen del Documento: Esto le permite a DocBits conocer el país de origen de un documento a través de expresiones regulares. Por ejemplo, si una expresión regular para un documento español contiene el término “Factura”. Si DocBits busca en un documento y encuentra este término, entonces sabría que el documento es de origen español y lo clasificaría como tal.

## **Acceso al Gestor de Regex**

Para encontrar esta función en DocBits, desde su Panel de Control, navegue a Configuración → Configuración Global → Tipos de Documento. Dentro de cada uno de los tipos de documentos creados, hay una opción de “Regex”.

## **Añadir y eliminar Regex**

Al hacer clic en “Regex”, se le llevará a este menú, que muestra los modelos de regex existentes que se han creado, así como un botón de “AÑADIR” para que cree nuevos modelos de regex.
