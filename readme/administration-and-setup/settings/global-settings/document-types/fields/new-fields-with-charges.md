# Nuevos Campos con Cargos


## Paso 1: Acceder a la Configuración de Campos

1. Navegar a Campos:

• Desde el panel principal, haga clic en la opción "Campos" ubicada en la barra lateral.

• Esto abrirá la página de Configuración de Campos donde puede gestionar los tipos de documentos y la configuración de reconocimiento.

2. Seleccionar Tipo de Documento:

• Bajo la sección "TIPOS DE DOCUMENTO", seleccione el tipo de documento específico para el cual desea añadir o modificar un campo.

## Paso 2: Añadir un Nuevo Campo

1. Abrir el Diálogo Añadir Campo:

• Haga clic en el botón "Crear Campo" ubicado en la sección respectiva (por ejemplo, Detalles de la Factura, Detalles del Pago, Orden de Compra).

2. Introducir Detalles del Campo:

• Aparecerá un cuadro de diálogo titulado "Detalles del campo del documento".

• Complete los detalles requeridos:

• Nombre: Ingrese el nombre del nuevo campo.

• Título: Ingrese un título descriptivo para el campo.

• Seleccionar tipo de campo: Elija el tipo de campo apropiado del menú desplegable.

3. Habilitar Elemento de Cargos:

• Si este campo está asociado con un cargo, marque la casilla "Habilitar elemento de cargos".

4. Seleccionar Mapeo de Costos:

• Al habilitar los cargos, aparecerá un menú desplegable.

• Seleccione el tipo de cargo apropiado de la lista (por ejemplo, ADD ON - ORIGIN, FUEL SURCHARGE, TOTAL FREIGHT).


## Paso 3: Guardar el Nuevo Campo

1. Guardar Configuración:

• Haga clic en el botón "GUARDAR" para añadir el nuevo campo con el mapeo de cargo especificado.

• Si necesita realizar cambios, haga clic en el botón "VOLVER" para regresar a la pantalla anterior.

2. Finalizar la Creación del Campo:

• Después de guardar, el nuevo campo aparecerá en la lista bajo el tipo de documento especificado.

• Asegúrese de que los ajustes de OCR y Match Score estén configurados según sea necesario para un reconocimiento preciso.

3. Completar la Configuración:

• Una vez que todos los campos deseados hayan sido añadidos y configurados, haga clic en el botón "Guardar Configuración" en la parte inferior de la página de Configuración de Campos para aplicar sus cambios.


## Notas Adicionales

• Campos Obligatorios:

• Si un campo es obligatorio, marque la casilla "OBLIGATORIO" junto al nombre del campo.

• Editar Campos Existentes:

• Para editar un campo existente, haga clic en el nombre del campo, actualice los detalles y guarde los cambios.

• Reasignar Grupo de Campos:

• Utilice la opción "Reasignar Grupo de Campos" para cambiar la agrupación de campos si es necesario.

• Configuración de Datos Maestros:

• Para una configuración avanzada, acceda a "Configuración de Datos Maestros" para gestionar la configuración general de campos y tipos de documentos.
