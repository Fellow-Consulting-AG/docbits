# Nouveaux champs avec frais


## Étape 1 : Accéder aux paramètres du champ

1. Naviguer vers Champs :

• Depuis le tableau de bord principal, cliquez sur l'option "Champs" située dans la barre latérale.

• Cela ouvrira la page Paramètres des champs où vous pourrez gérer les types de documents et les paramètres de reconnaissance.

2. Sélectionner le type de document :

• Sous la section "TYPES DE DOCUMENTS", sélectionnez le type de document spécifique pour lequel vous souhaitez ajouter ou modifier un champ.

## Étape 2 : Ajouter un nouveau champ

1. Ouvrir la boîte de dialogue Ajouter un champ :

• Cliquez sur le bouton "Créer un champ" situé dans la section correspondante (par exemple, Détails de la facture, Détails du paiement, Bon de commande).

2. Saisir les détails du champ :

• Une boîte de dialogue intitulée "Détails du champ de document" apparaîtra.

• Remplissez les détails requis :

• Nom : Saisissez le nom du nouveau champ.

• Titre : Saisissez un titre descriptif pour le champ.

• Sélectionner le type de champ : Choisissez le type de champ approprié dans le menu déroulant.

3. Activer l'élément Frais :

• Si ce champ est associé à des frais, cochez la case "Activer l'élément de frais".

4. Sélectionner le mappage des coûts :

• Une fois les frais activés, un menu déroulant apparaîtra.

• Sélectionnez le type de frais approprié dans la liste (par exemple, ADD ON - ORIGIN, FUEL SURCHARGE, TOTAL FREIGHT).


## Étape 3 : Enregistrer le nouveau champ

1. Enregistrer les paramètres :

• Cliquez sur le bouton "ENREGISTRER" pour ajouter le nouveau champ avec le mappage de frais spécifié.

• Si vous devez apporter des modifications, cliquez sur le bouton "RETOUR" pour revenir à l'écran précédent.

2. Finaliser la création du champ :

• Après l'enregistrement, le nouveau champ apparaîtra dans la liste sous le type de document spécifié.

• Assurez-vous que les paramètres OCR et Match Score sont configurés comme nécessaire pour une reconnaissance précise.

3. Terminer la configuration :

• Une fois tous les champs souhaités ajoutés et configurés, cliquez sur le bouton "Enregistrer les paramètres" en bas de la page Paramètres des champs pour appliquer vos modifications.


## Notes supplémentaires

• Champs obligatoires :

• Si un champ est obligatoire, cochez la case "OBLIGATOIRE" à côté du nom du champ.

• Modifier des champs existants :

• Pour modifier un champ existant, cliquez sur le nom du champ, mettez à jour les détails et enregistrez les modifications.

• Réassigner le groupe de champs :

• Utilisez l'option "Réassigner le groupe de champs" pour modifier le regroupement des champs si nécessaire.

• Paramètres des données de référence :

• Pour une configuration avancée, accédez aux "Paramètres des données de référence" pour gérer les paramètres globaux des champs et des types de documents.
