# Nuovi Campi con Oneri


## Passaggio 1: Accedere alle Impostazioni del Campo

1. Navigare verso i Campi:

• Dalla dashboard principale, fare clic sull'opzione "Campi" situata nella barra laterale.

• Questo aprirà la pagina Impostazioni campo dove è possibile gestire i tipi di documenti e le impostazioni di riconoscimento.

2. Selezionare il Tipo di Documento:

• Sotto la sezione "TIPI DI DOCUMENTO", selezionare il tipo di documento specifico per cui si desidera aggiungere o modificare un campo.

## Passaggio 2: Aggiungere un Nuovo Campo

1. Aprire la finestra di dialogo Aggiungi campo:

• Fare clic sul pulsante "Crea campo" situato nella rispettiva sezione (ad esempio, Dettagli fattura, Dettagli pagamento, Ordine di acquisto).

2. Inserire i dettagli del campo:

• Apparirà una finestra di dialogo intitolata "Dettagli campo documento".

• Compilare i dettagli richiesti:

• Nome: Inserire il nome del nuovo campo.

• Titolo: Inserire un titolo descrittivo per il campo.

• Seleziona tipo di campo: Scegliere il tipo di campo appropriato dal menu a discesa.

3. Abilitare l'elemento Oneri:

• Se questo campo è associato a un onere, selezionare la casella "Abilita elemento oneri".

4. Selezionare la mappatura dei costi:

• All'abilitazione degli oneri, apparirà un menu a discesa.

• Selezionare il tipo di onere appropriato dall'elenco (ad esempio, ADD ON - ORIGIN, FUEL SURCHARGE, TOTAL FREIGHT).


## Passaggio 3: Salvare il Nuovo Campo

1. Salvare le impostazioni:

• Fare clic sul pulsante "SALVA" per aggiungere il nuovo campo con la mappatura degli oneri specificata.

• Se è necessario apportarvi modifiche, fare clic sul pulsante "INDIETRO" per tornare alla schermata precedente.

2. Finalizzare la creazione del campo:

• Dopo il salvataggio, il nuovo campo apparirà nell'elenco sotto il tipo di documento specificato.

• Assicurarsi che le impostazioni OCR e Match Score siano configurate come necessario per un riconoscimento accurato.

3. Completare la configurazione:

• Una volta aggiunti e configurati tutti i campi desiderati, fare clic sul pulsante "Salva impostazioni" in fondo alla pagina Impostazioni campo per applicare le modifiche.


## Note aggiuntive

• Campi obbligatori:

• Se un campo è obbligatorio, selezionare la casella "OBBLIGATORIO" accanto al nome del campo.

• Modificare i campi esistenti:

• Per modificare un campo esistente, fare clic sul nome del campo, aggiornare i dettagli e salvare le modifiche.

• Riassegnare il gruppo di campi:

• Utilizzare l'opzione "Riassegna gruppo di campi" per modificare il raggruppamento dei campi se necessario.

• Impostazioni dei dati anagrafici:

• Per la configurazione avanzata, accedere a "Impostazioni dati anagrafici" per gestire le impostazioni generali del campo e del tipo di documento.
