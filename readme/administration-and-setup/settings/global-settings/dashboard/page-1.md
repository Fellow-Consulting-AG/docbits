![](https://lh7-us.googleusercontent.com/aEOmeKPrkuBe0aSRJKDi5QZ1UBaTjuq0xdfRkJWv78k6XZ42QKYccjaTRXVjrNPmn86-vTeKSNWmWuReYvn-ieD3A6aMoi_sVKTKSKMy992Dz-YXVmOSIHRsTTHeU50b4nwyLbwVWlG9F_3D6fO-GPU)

Per creare un Filtro Personalizzato, seleziona PERSONALIZZATO nello "Stile filtro stato"

Sotto di esso, puoi selezionare i vari stati in cui un documento può trovarsi per creare il tuo filtro personalizzato. Una volta premuto APPLICA, questo filtro personalizzato sarà attivo sulla Dashboard.
