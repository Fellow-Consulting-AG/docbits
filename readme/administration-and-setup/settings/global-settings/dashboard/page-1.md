Navigieren Sie zum Abschnitt Filter

![](https://lh7-us.googleusercontent.com/aEOmeKPrkuBe0aSRJKDi5QZ1UBaTjuq0xdfRkJWv78k6XZ42QKYccjaTRXVjrNPmn86-vTeKSNWmWuReYvn-ieD3A6aMoi_sVKTKSKMy992Dz-YXVmOSIHRsTTHeU50b4nwyLbwVWlG9F_3D6fO-GPU)

Um einen benutzerdefinierten Filter zu erstellen, wählen Sie CUSTOM im "Status-Filterstil".

Darunter können Sie die verschiedenen Status auswählen, die ein Dokument haben kann, um Ihren benutzerdefinierten Filter zu erstellen. Sobald Sie auf APPLY drücken, wird dieser benutzerdefinierte Filter auf dem Dashboard aktiv sein.

**Dashboard**

Wählen Sie ERWEITERTE EINSTELLUNGEN
