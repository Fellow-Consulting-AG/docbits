# Kontrol Paneli Verilerini Dışa Aktar

Bu modül, DocBits kontrol panelindeki istatistiksel verilerin ve raporların dışa aktarılmasını sağlar. Bu veriler, iş süreçlerini analiz etmek, performans takibi yapmak ve özel raporlar oluşturmak için kullanılabilir.

## Dışa Aktarılabilir Veriler

* **İşlem Hacimleri**: Belirli bir dönemde işlenen belge sayısı.
* **Kullanıcı Performansı**: Kullanıcı başına işleme süreleri ve doğruluk oranları.
* **Hata Oranları**: Otomasyon hataları ve manuel düzeltme gerektiren belgeler.
* **Belge Türü Dağılımı**: İşlenen belgelerin türlerine göre dağılımı.

## Yapılandırma

1. **Format Seçimi**: Verilerin hangi formatta dışa aktarılacağını seçin (örneğin CSV, Excel, JSON).
2. **Zamanlama**: Verilerin otomatik olarak belirli aralıklarla (örneğin günlük, haftalık) dışa aktarılmasını planlayın.
3. **Hedef**: Dışa aktarılan verilerin nereye gönderileceğini belirleyin (örneğin E-posta, FTP, Dosya Sistemi).
