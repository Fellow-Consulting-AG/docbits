# IDM ACL Güncelleyici

## Genel Bakış

ACL Güncelleyici modülü, DocBits içindeki Erişim Kontrol Listelerini (ACL'ler) güncellemek ve yönetmekten sorumludur. Bu, belge erişimi ve işlenmesi için uygun izinlerin ve güvenlik önlemlerinin yürürlükte olmasını sağlar.

## Aktivasyon Talimatları

ACL Güncelleyici modülünü aktive etmek için şu adımları izleyin:

1. **Ayarlar**'a gidin.
2. **Belge İşleme**'yi seçin.
3. **Modül**'ü seçin.
4. **Belge Türü**'ne gidin.
5. Karşılık gelen kaydırıcıyı etkinleştirerek **IDM ACL Güncelleyici**'yi aktive edin.

Modül aktive edildiğinde, kullanılacak bir **ION Eşleme Dosyası** seçebilirsiniz.

### ACL Bilgisi Ekleme

ACL bilgisi eklemek için **Ekle** (Add) düğmesine tıklayın. Ardından aşağıdaki ayrıntıları belirtebilirsiniz:

* **IDM Belge Türü**
* **Mevcut IDM ACL Durumu**
* **Yeni IDM ACL Durumu**
