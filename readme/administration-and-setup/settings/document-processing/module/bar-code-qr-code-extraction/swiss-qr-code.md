# Zwitserse QR-Code

## QR-Code Extractie Functie

De QR-Code Extractie functie maakt automatische extractie en matching van QR-code gegevens van documenten mogelijk, zoals Zwitserse facturen, met opgeslagen informatie zoals leveranciersgegevens. Het activeren van deze functie stelt u in staat om gegevensverwerking te stroomlijnen, essentiële informatie op te halen zonder handmatige invoer.

### Functieoverzicht

QR-Code Extractie detecteert QR-codes binnen geüploade documenten, extraheert de gecodeerde gegevens en matcht deze met bestaande records. Dit geautomatiseerde proces bevordert efficiënte documentafhandeling, met name gunstig voor facturen en andere gestandaardiseerde documenten die QR-codes bevatten.

#### Belangrijkste Voordelen

* **Geautomatiseerde Gegevensvastlegging**: Vermindert handmatige gegevensinvoer door automatisch informatie uit QR-codes te extraheren.
* **Verbeterde Nauwkeurigheid**: Minimaliseert menselijke fouten door QR-code informatie direct te matchen met opgeslagen gegevens.
* **Tijdsefficiëntie**: Versnelt documentverwerking door repetitieve taken te elimineren.

***

### Hoe QR-Code Extractie In te Schakelen

Volg deze stappen om de QR-Code Extractie functie in te schakelen:

1. **Navigeer naar Instellingen**:
   * Ga vanuit het Dashboard naar **Instellingen**.
   * Selecteer **Documentverwerking** en kies vervolgens **Module**.
2. **Schakel de Functie In**:
   * Scroll naar beneden om de optie **QR-Code Extractie** te vinden.
   * Gebruik de schuifregelaar om QR-Code Extractie in te schakelen.
