# Doc Script

**Doc Script**, belge işleme sırasında özel mantık ve otomasyon uygulamak için kullanılan güçlü bir komut dosyası aracıdır. Python veya JavaScript benzeri bir sözdizimi kullanarak karmaşık veri doğrulama, dönüştürme ve iş akışı kuralları oluşturabilirsiniz.

## Kullanım Alanları

* **Gelişmiş Doğrulama**: Standart kuralların ötesinde, özel iş mantığına dayalı veri doğrulaması yapın.
* **Veri Dönüştürme**: Çıkarılan verileri formatlayın, birleştirin veya hesaplamalar yapın.
* **Koşullu Yönlendirme**: Belge içeriğine göre belgeleri farklı iş akışlarına yönlendirin.
* **Harici API Entegrasyonu**: İşleme sırasında harici sistemlerden veri çekin veya veri gönderin.

## Yapılandırma

1. **Komut Dosyası Düzenleyici**: Komut dosyalarınızı yazmak ve test etmek için yerleşik düzenleyiciyi kullanın.
2. **Tetikleyiciler**: Komut dosyasının ne zaman çalışacağını belirleyin (örneğin, İçe Aktarmadan Sonra, Doğrulamadan Önce, Dışa Aktarmadan Önce).
