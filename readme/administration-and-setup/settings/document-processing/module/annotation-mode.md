# Açıklama Modu

**Açıklama Modu**, DocBits'in belge işleme sırasında ek açıklamaları ve notları nasıl ele aldığını belirler.

## Ayarlar

<figure><img src="../../../../../.gitbook/assets/module_annotation_mode.png" alt=""><figcaption></figcaption></figure>

**Etkin**: Açıklama Modunu etkinleştirmek için bu seçeneği açın. Etkinleştirildiğinde, kullanıcılar belgeleri işlerken açıklamalar ekleyebilir, düzenleyebilir ve görüntüleyebilir.

**Devre Dışı**: Açıklama Modunu devre dışı bırakmak için bu seçeneği kapatın.
