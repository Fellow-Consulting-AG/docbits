# Doc Scan

**Doc Scan** modülü, fiziksel belgelerin doğrudan DocBits arayüzü üzerinden taranmasını ve içe aktarılmasını sağlar.

## Özellikler

* **Doğrudan Tarama**: TWAIN veya WIA uyumlu tarayıcılar kullanarak belgeleri doğrudan tarayın.
* **Görüntü İyileştirme**: Taranan görüntüleri otomatik olarak iyileştirin (örneğin, eğrilik düzeltme, gürültü giderme).
* **Toplu Tarama**: Birden fazla sayfayı tek bir belge olarak veya ayrı belgeler olarak tarayın.

## Yapılandırma

1. **Tarayıcı Seçimi**: Kullanılacak varsayılan tarayıcıyı seçin.
2. **Tarama Profilleri**: Farklı belge türleri için (örneğin Renkli, Siyah Beyaz, Yüksek Çözünürlük) tarama profilleri oluşturun.
